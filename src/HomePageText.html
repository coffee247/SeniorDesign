R-BIMS

The Replacement Ballistic Information Management System (R-BIMS),
is a second generation Ballistic Information Management System, designed and created by students
at Virginia Commonwealth University (VCU) as a capstone project during their senior year.

How to edit THIS help page text, (an HTML doccument) ...

  1. Edit the HTML source of the file HomePageText.html which is in the src directory
  2. from the command line issue the following command (format the code for consumption by this software)

    $ sudo pyrcc5 -o BIMSresources.py BIMS_Resources.qrc

IMPORTANT:

  • You must be in the same directory as the HomePageText.html file when the above command is issued.
  • You must know the root user password.
