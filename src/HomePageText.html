R-BIMS

The Replacement Ballistic Information Management System (R-BIMS),
is a second generation Ballistic Information Management System,

designed and created by: James Stallings (https://github.com/coffee247/SeniorDesign)

at Virginia Commonwealth University (VCU) as a capstone project during his senior year.

  1. Edit the HTML source of the file HomePageText.html which is in the src directory

    $ sudo pyrcc5 -o BIMSresources.py BIMS_Resources.qrc

IMPORTANT:

  • You must be in the same directory as the HomePageText.html file when the above command is issued.
  • You must know the root user password.
